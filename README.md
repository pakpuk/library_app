# 📚 Library Management App

A mobile app to manage book collections, lending, and user accounts.

## ✨ Features
- 📖 Browse and search library catalog  
- 👤 User registration & login  
- 📅 Borrow & return books  
- 📝 Keep track of reading history  
- 🔔 Notifications for due dates  

## 🛠 Tech Stack
- Flutter (Dart)  
- Firebase (backend)
